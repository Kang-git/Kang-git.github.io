vscode常用快捷键及插件

Apr 29, 2019点击

哈哈，弄好博客也有好多天了，今天（五一假期的第三天）终于有空来了写一篇文章了，初次写一篇正式的博客文章，多少感觉有点手忙脚乱，不知从何下手，后来想想，既然无从下手，就干脆当成引导自己学习的技术的日记吧，最近一直在频繁的使用VS Code，索性就直接来记录下自己在vscode上安装的一些插件，顺带安利一下在用的主题,文章最后写一些常用的快捷键，可能你有些也是自己在学习使用中，毕竟自己也不可能一次性记住很多快捷键，同时也便于自己查找快捷键时翻看。
好了，下面进入正文部分。-_-

VS Code插件（前端为主）

1.Atom One Dark Theme:一个深色的主题插件。代码高亮明显
2.Beautify:一个用来美化代码的插件
3.Chinese(Simplified)Language Pack for Visual Studio Code:简体中文语言插件(虽然可能推荐这个插件有点罪恶，且无助于英语能力的提升)但是对于刚开始接触VS Code的人来说，可能还是需要将语言汉化一下（不安装该插件只改设置汉化不会成功的）。
4.Color Highlight:颜色高亮显示
5.Debugger for Chrome:故名思义
6.Gitlens：使用git管理的插件
7.Hackers Haze Theme：黑客主题
8.HTML CSS Support：HTML和CSS一些语法的支持插件
9.Image preview：图片预览
10.Material Theme：一个风格比较靓丽的主题插件

快捷键

1.单行注释： ctrl+/
2.取消单行注释： ctrl+u
3.多行注释： alt+shift+A
4.向上或向下复制当前行： shift+alt+up/down
5.删除当前行： shift+ctrl+k
6.代码格式化： shift+alt+f
7.新建一个窗口： ctrl+shift+n
8.显示比例缩放： ctrl+(+/-)
9.跳转到代码指定行： ctrl+g
10.选中代码： shift+鼠标左键
11.回到顶部： ctrl+home
12.回到底部： ctrl+end
